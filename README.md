# Blinkit Homepage Clone

A responsive, pixel-perfect front-end clone of the **Blinkit** (formerly Grofers) web interface. This project focuses on replicating the core structure of the e-commerce application, including the navigation header, promotional banners, multi-row product feeds, and a comprehensive footer section.

## 🚀 Features

- **Semantic HTML5 Structure:** Built using proper semantic tags (`<header>`, `<section>`, `<footer>`) for clean code and better SEO mapping.
- **Flexbox Layout Architecture:** Leverages modern CSS Flexbox for fluid alignment, spacing, and grid formations across rows.
- **Fully Responsive Design:** Implements CSS Media Queries to seamlessly shift from a desktop-wide grid to a mobile-optimized layout.
- **Mobile-First App Experience:** On small screens, the desktop navigation transforms into a vertical layout, and key utility buttons lock to a sticky bottom tray for mobile ease-of-use.
- **Vector Graphics:** Uses inline SVGs to ensure logos maintain crisp resolution across high-density displays.

---

## 🛠️ Technologies Used

* **HTML5:** Used for creating the semantic structure of the page (header, sections, footer) and form elements.
* **CSS3:** Used for all styling, including layout alignment (Flexbox) and mobile responsiveness (Media Queries).

---

## 📱 Layout Structure Overview

The codebase is organized into four logical page segments:

1. **Header:** Features the vector Blinkit logo, dynamic delivery time notification, interactive search field, user login, and cart utility buttons.
2. **Promotional Area:** Display grids featuring localized category banners (Pharmacy, Pet Care, Baby Care).
3. **Product Collections:** Modular grid components showcasing item details (Amul Dairy goods, snacks, etc.) with functional pricing badges and "Add" button states.
4. **Footer navigation:** Clean columns organizing customer utility links, partner resources, mobile application store buttons, and social media handles.
